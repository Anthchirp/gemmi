$ gemmi sf2map -h
Usage:
  gemmi sf2map [options] INPUT_FILE MAP_FILE

INPUT_FILE must be either MTZ or mmCIF with map coefficients.

By default, coefficients for a 2mFo-DFc map are used.
If "-d" is specified - for a mFo-DFc map.
Coefficients for a normal map are expected in MTZ columns FWT/PHWT or
2FOFCWT/PH2FOFCWT, or under mmCIF tags _refln.pdbx_FWT/pdbx_PHWT.
For a difference map - in MTZ columns DELFWT/PHDELWT or FOFCWT/PHFOFCWT,
or under mmCIF tags _refln.pdbx_DELFWT/pdbx_DELPHWT

Options:
  -h, --help           Print usage and exit.
  -V, --version        Print version and exit.
  --verbose            Verbose output.
  -d, --diff           Use difference map coefficients.
  --section=NAME       MTZ dataset name or CIF block name
  -fCOLUMN             F column (MTZ label or mmCIF tag).
  -pCOLUMN             Phase column (MTZ label or mmCIF tag).
  -g, --grid=NX,NY,NZ  Grid sampling.
  -G                   Print grid size that would be used and exit.
