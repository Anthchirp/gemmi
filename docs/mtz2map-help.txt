$ gemmi mtz2map -h
Usage:
  gemmi mtz2map [options] INPUT_FILE MAP_FILE
INPUT_FILE must be either MTZ or mmCIF with map coefficients.
Options:
  -h, --help           Print usage and exit.
  -V, --version        Print version and exit.
  --verbose            Verbose output.
  -d, --diff           Use data from columns (PH)DELFWT or (PH)FOFCWT.
  -fCOLUMN             F label in the MTZ file.
  -pCOLUMN             Phase label in the MTZ file.
  -g, --grid=NX,NY,NZ  Grid sampling.
